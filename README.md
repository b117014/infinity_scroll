# Infinity Scroll

## Tech Stack

##### Frontend
    1. React.js
    4. Html
    5. css

##### Backend
    1. Node.js
    2. Mongodb
    3. Express.js

# Installation guide for Backend

- clone this repo:

```
git clone repo
```

- Install necessary dependencies:

```
npm install
```

- Run the server:

```
node index.js
```
# Installation guide for Frontend


- Install necessary dependencies:


```
npm install


- Run the server:

```
npm start
```
